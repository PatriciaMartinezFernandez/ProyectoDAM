1. Sentencia IF-ELSE


La sentencia if-else en Java permite implementar estructuras condicionales simples. Su formato básico es el siguiente:


        if (condicion) {
            //acciones si se cumple la condicion
        } else {
            //acciones si no se cumple la condicion
        }

El bloque else no es obligatorio y solo se añade si se desea realizar alguna acción específica cuando la condición no se cumple.

3. Sentencia SWITCH


La sentencia switch en Java ofrece una forma eficiente de evaluar múltiples casos sin pasar por todos ellos. Aunque limitada a condiciones de igualdad, funciona con tipos primitivos y cadenas. Existen dos versiones: como sentencia, para ejecutar bloques de código, y como expresión, para asignar directamente resultados.


        switch (valor) {
            case 0: calcular los impuestos para declarante soltero;
                break;
            case 1: calcular los impuestos para declarante casado conjuntamente;
                break;
            case 2: calcular los impuestos para declarante casado por separado;
                break;
            case 3: calcular los impuestos para el cabeza de familia;
                break;
        }

4. Depuración y depurador


Los errores lógicos se llaman errores, y corregirlos se conoce como depuración. Un método común es reducir la parte del programa con errores, ya sea rastreándolo manualmente o mediante impresiones de variables. Este enfoque es útil para programas simples, pero para proyectos grandes, es más efectivo usar un depurador. Un depurador facilita la ejecución paso a paso, el seguimiento de métodos, la fijación de puntos de interrupción y la visualización de variables y la pila de llamadas.
