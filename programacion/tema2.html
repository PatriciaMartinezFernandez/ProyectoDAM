1. Variables


Una variable es un dato que se almacenan memoria y se utiliza para realizar procesamientos específicos. Por ejemplo, para el cálculo del área de un triángulo, se requieren tres variables: base, altura y resultado. Cada variable se define mediante un identificador, un tipo de dato y un valor actual.

4. Constantes


Una variable que mantiene un valor fijo se denomina constante. Para definir este tipo de valores inmutables, se utiliza la palabra clave final antes del tipo de dato, seguido del identificador, que debe de escribirse en mayúsculas, y el valor asignado. Por ejemplo, final double PI = 3.14159;


5. Scanner


Hasta ahora, hemos manejado valores literales, es decir, datos escritos directamente en el código. Para permitir que el usuario introduzca valores por teclado utilizamos la clase Scanner en Java, que forma parte del paquete java.util.


        var sc = new Scanner(System.in);
        //Resto de código
        sc.close;

La clase Scanner proporciona mecanismos para leer distintos tipos de datos, como números con sc.nextInt() y cadenas de caracteres con sc.nextLine(). Estos métodos se utilizan a la derecha de una asignación. Por ejemplo, int numero = sc.nextInt();

7. Operadores de incremente y decremento


Operador Nombre Descripción Ejemplo
++var Preincremento Incrementa var en 1 y usa el nuevo valor de var en la sentencia int j = ++i;
var++ Posincremento Incrementa var en 1 pero usa el valor original en la sentencia int j = i++;
--var Predecremento Decrementa var en 1 y usa el nuevo valor de var en la sentencia int j = --i;
var-- Posdecremento Decrementa var en 1 pero usa el valor original en la sentencia int j = i--;
