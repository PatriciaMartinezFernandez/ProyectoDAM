1. Programas


Los programas informáticos, también llamados software, consisten en instrucciones para el ordenador. Sin programas, un ordenador no es más que una máquina vacía. EL ordenador no entiende el lenguaje humano, por lo que se utilizan lenguajes de programación para comunicarnos con ellas.


Funciones básicas del ordenador:

  • Operaciones numéricas: Sumar, restar, multiplicar y dividir dos valores numéricos.

  • Comparación de valores: Verificar la igualdad entre dos valores y determinar si el primer valor es mayor o si el segundo lo es.

  • Almacenamiento y recuperación de información: Almacenar información en la memoria del ordenador y/o en dispositivos externos.

La combinación de estas operaciones permite la creación y ejecución de programas.


2. Algoritmos


Un algoritmo es un conjunto ordenado de pasos que se deben realizar para llevar a cabo una acción específica.


Por ejemplo, hacer una tortilla:

  1. Cascar dos huevos en un plato hondo.

  2. Añadir sal a gusto.

  3. Batir los huevos enérgicamente con un tenedor.

  4. Encender el fuego de la cocina.

  5. Echar aceite en la sartén.

  6. Calentar la sartén con el aceite.

  7. Verter los huevos batidos en la sartén caliente.

  8. Esperar mientras se fríe la base de los huevos.

  9. Doblar al centro la base de los huevos batidos.

  10. Esperar a que se termine de freír.

  11. Sacar la tortilla de la sartén y servir.

Características de los algoritmos:

  • Finito: Debe finalizar su ejecucion en algún punto.
  • Legibilidad: Debe estar escrito de tal forma que sea fácil de leer y entender.
  • Modificabilidad: Debe poderse realizar fácilmente modificaciones y actualizaciones, para adaptarse a nuevos requisitos.
  • Eficiencia: Debe minimizar el espacio de memoria usado y el tiempo de ejecución empleado.
  • Modularidad: El programa puede estar dividido en módulos, cada uno de los cuales realiza una parte del problema.
  • Estructuración: Comprende todas las características anteriores. Como consecuencia de una mayor estructuración, resulta más fácil: Leerlo, modificarlo y eliminar las redundancias.

3. Lenguajes de programación


Lenguaje máquina: El lenguaje de máquina es un conjunto de instrucciones primitivas en código binario. Programar en este lenguaje es tedioso y difícil de leer o modificar. Por ejemplo, la instrucción binaria para sumar dos números podría ser "1101101010011010".


Lenguaje ensamblador: Se crearon lenguajes ensambladores para simplificar la programación. Aunque los ordenadores no pueden comprender directamente el lenguaje ensamblador, se utiliza un programa llamado ensamblador para convertir programas escritos en este lenguaje a código de máquina. Por ejemplo, para sumar dos números, se escribe la instrucción como: ADDF3 R1, R2, R3


Lenguaje de alto nivel: Los lenguajes de alto nivel, generalmente en inglés, son fáciles de aprender y programar. Una instrucción en este tipo de lenguaje para calcular el área de un círculo con radio 5 es: area = 5 * 5 * 3.1415;

Los lenguajes de alto nivel más conocidos son: C, C++, C#, Java, Python...


4. Interpretación / Compilación de código fuente


Un programa escrito en lenguaje de alto nivel se llama código fuente, para ejecutarlo en un ordenador se necesita traducirlo a código de máquina. Esta traducción puede hacerse mediante un intérprete o compilador.


Interpretación de código fuente: Un intérprete lee una instrucción del código fuente, la traduce al código de máquina y la ejecuta inmediatamente. Es importante destacar que una instrucción del código fuente puede traducirse en varias instrucciones de la máquina.


Compilación de código fuente: Un compilador traduce todo el código fuente en un archivo de código máquina y el archivo de código máquina se ejecuta completo.


5. ¿Por qué Java?


Java es un lenguaje de programación de propósito general que desempeña un papel crucial en el desarrollo de aplicaciones para Internet. Algunas características claves incluyen:


  1. Versatibilidad de Aplicaciones: Desarrollo para servidores, computadoras de escritorio y dispositivos portátiles.

  2. Importancia en Internet: Java se destaca como el lenguaje de programación para aplicaciones web.

En cuanto a la historia de Java, fue desarrollado por James Gosling y Sun Microsystems, lanzándose en 1995. Entre sus características se encuentran la simplicidad, orientación a obketos, distribución, interpretación, robustez, seguridad, neutralidad en la arquitectura, portabilidad, rendimiento, capacidad multiproceso y dinamismo. Java sigue siendo una parte esencial del panorama informático actual.
